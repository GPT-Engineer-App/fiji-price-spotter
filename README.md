# fiji-price-spotter

an app that compares prices against various supermarket, and retailers in Fiji, displays weekly deals that are on special or according to the price comparison cheaper then last week.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/fiji-price-spotter.git
cd fiji-price-spotter
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
